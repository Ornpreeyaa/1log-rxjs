1log-rxjs

1log plugin for logging RxJS observables. In addition to logging nexts, errors, and completions, this plugin creates log messages when an observable is created, subscribed, or unsubbed. When an observable is subscribed multiple times, numbered badges let you attribute each next, error, completion, or unsubscription to a specific subscription. The plugin can also be used in combination with Jest's snapshots feature to test observables by creating snapshots of log messages.

Please refer to the GitHub README for full documentation.
